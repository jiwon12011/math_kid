# 수리수리 도감 🔮🐸

> 흑백 동물 그림이 구구단을 맞힐 때마다 **"수리수리마수리~"** 하고 색이 살아나고, 완성하면 내 **도감**에 들어가는 아이들용 구구단 학습 앱.

7~10세 아이들을 위한 귀여운 크레용 스타일 사칙연산 학습 PWA. **MVP가 완성된 상태**이며(바닐라 HTML/CSS/JS, 빌드 없음), 이 문서는 제품의 컨셉·기능·설계 의도를 함께 담은 소개 문서입니다.

---

## 한 줄 컨셉

**"수(數) + 수리수리마수리(마법) + 도감(수집)"** — 정답을 맞히면 흑백 그림에 색이 마법처럼 살아나 동물 도감을 완성하는 구구단 앱.

## 왜 만들었나

아동용 앱을 만들어보고 싶던 차에, 친구 아이가 마침 구구단을 배우기 시작했다. 그래서 어린이 학습 앱들을 둘러보다 두 가지가 계속 걸렸다 — **광고**, 그리고 **인터넷이 없으면 멈춘다는 점**.

- 무료 어린이 앱은 대부분 광고로 돌아간다. 아이가 쓰다가 전면 광고를 잘못 눌러 엉뚱한 데로 넘어가기 쉽다. 적어도 **아이가 보는 화면은 광고로 방해받지 않아야** 한다고 생각했다.
- 요즘은 너무 많은 앱이 인터넷 연결을 전제로 한다. 하지만 아이가 앱을 쓰는 곳은 차 안·비행기·데이터 안 터지는 곳이 더 많다. **인터넷 없이 그냥 켜면 되는** 게 아이용엔 더 맞다고 봤다.

그래서 **무광고·오프라인**을 첫 원칙으로 두고, 어차피 반복으로 외우는 구구단을 "맞히면 그림이 색칠되는" 수집형 놀이로 풀었다.

> 한 줄로: **광고 없이, 인터넷 없이도 아이 혼자 안심하고 쓸 수 있는 구구단 앱을 만들고 싶었다.**

## 타깃 & 플랫폼

- **타깃**: 7~10세 (구구단을 배우는 시기)
- **플랫폼**: PWA (웹, 설치형) — 앱스토어 심사·개인정보 부담을 낮추고 빠르게 검증
- **범위**: 구구단 중심 (추후 사칙연산으로 확장 가능)

## 차별성

| 보통의 구구단 앱 | 수리수리 도감 |
|---|---|
| 정답 → 점수·별 적립 | 정답 → **흑백 그림이 한 칸씩 컬러로 복귀** ("수리수리!") |
| 끝없는 문제 풀이 | **12문제 = 그림 1장 완성**이라는 명확한 결승선 |
| 보상이 추상적(숫자) | 보상이 **소장 가능한 도감 50종** — 수집 욕구를 자극 |
| 텍스트 중심 | **음성 안내**로 읽기 전 아이도 혼자 플레이 |
| 똑같은 문제 무작위 | **약점 자동 감지** → 틀리는 단을 집중 추천 |
| 광고·계정·결제 | **무광고·무계정·로컬 저장·오프라인** |

**가장 뾰족한 한 가지** — 학습의 보상이 '점수'가 아니라 **'내가 색칠해 완성한 작품'**이다. 끝나고 남는 게 트로피 숫자가 아니라 자랑할 수 있는 컬렉션.

**기술적 영리함** — 컬러 이미지 1장만 만들면 코드가 자동으로 흑백 필터(`grayscale`)를 씌우고, 정답마다 4×3 마스크를 한 칸씩 걷어내 색을 되살린다. 동물 50종을 위해 흑백/컬러를 따로 그릴 필요 없이 **컬러 1장만 준비하면 어떤 그림에도 적용**된다 (제작 비용·확장성 이점).

## 세계관 & 비주얼

- **세계관 = "스케치북"**: 앱 전체가 내 스케치북. 상단에 스프링이 달린 노트 느낌.
- **비주얼 = 크레용 일러스트톤**: 손 그린 듯 따뜻하고 소박한 결.
- **팔레트**: `#F4A460` `#87CEEB` `#98FB98` `#FFB6C1` `#F5DEB3`

## 학습 범위 (사칙연산)

놀기에 들어가면 **연산 종류(➕덧셈 ➖뺄셈 ✖️곱셈 ➗나눗셈)** 를 고르고(여러 개 선택 가능), 곱셈은 **몇 단**까지 추가로 고른다. 덧셈·뺄셈·나눗셈은 난이도(쉬움/보통/어려움)에 따라 수 범위가 달라진다.

## 음성 안내

읽기 전·저학년 아이를 위해 **Web Speech API(ko-KR)** 로 문제를 읽어준다("7 곱하기 8은?"). 문제 보드의 🔈 버튼으로 다시 듣고, 정답/완성 시 음성 칭찬. 설정에서 켜고 끌 수 있다(아이용).

## 핵심 루프

```
범위 선택 (연산 종류 + 곱셈 단)
   ↓
스프링 스케치북에 크레용 손글씨로 구구단 문제 등장
   ↓
정답 1개 = 흑백 동물 그림이 4×3 격자로 한 칸씩 컬러 복귀 ("수리수리!")
   ↓
정답 12개 = 이미지 완성 ("마수리 완성!")
   ↓
완성된 컬러 동물 그림이 도감에 수록
```

- **패치 방식 A (격자)**: 12칸 = 4×3 격자. 정답마다 1칸씩 컬러 복귀. 어떤 그림에도 자동 적용.
- **페이싱**: 그림 1장 = 정답 12개.
- **구현 메모**: 컬러 이미지 1장만 준비 → 흑백은 코드 `grayscale` 필터로 자동 → 컬러 위에 패치 마스크를 덮고 정답마다 한 칸씩 걷어냄. (선화/컬러를 따로 생성하지 않음)

## 도감: 동물 50종 (5세트 × 10종)

희귀도 비율 — 흔함 60% / 특별 30% / ✨반짝이 10%. 반짝이는 어린이날·생일 등 한정 해금.

| 세트 | 흔함 | 특별 | ✨반짝이 |
|---|---|---|---|
| 🚜 농장 | 병아리·돼지·젖소·양·염소·오리 | 당나귀·칠면조·토끼 | 황금 닭 |
| 🌳 숲속 | 다람쥐·고슴도치·토끼·부엉이·사슴·청설모 | 여우·너구리·오소리 | 흰 사슴 |
| 🌊 바다 | 물고기·게·불가사리·문어·새우·거북이 | 해마·복어·가오리 | 무지개 물고기 |
| 🌴 정글 | 원숭이·앵무새·개구리·나비·카멜레온·아기고릴라 | 나무늘보·투칸·재규어 | 황금 개구리 |
| ❄️ 극지방 | 펭귄·북극곰·바다표범·눈올빼미·순록·빙하게 | 북극여우·벨루가·레밍 | 오로라 펭귄 |

**세트 완성 보너스**: 1세트=스티커 / 3세트=도감 커버 프레임 / 전체 50종=「마스터 도감사」 칭호.

| 탭 | 역할 |
|---|---|

## 아동 앱 게이트 (지킬 것)

- 개인정보 최소화 — MVP는 계정 없이 로컬 저장.
- 큰 둥근 터치 타깃(56×64px), 충분한 간격.
- 고대비(WCAG AA+), 색만으로 정보 전달 금지(색 + 아이콘/소리).
- 읽기 전 아이 대응 — 아이콘 + 음성 병행.
- `prefers-reduced-motion` 존중, 과자극 모션·강한 사운드 자제.
- 실패는 벌점이 아니라 격려 톤.
- 부모용 항목(기록·난이도)은 부모 게이트 뒤에.

## 설계하며 고민한 것

위 게이트가 "지킬 규칙"이라면, 아래는 그 규칙을 정한 **이유**다.

- **A. 발달 단계에 맞춤 (7~10세)** — 읽기 전 아이를 위해 모든 정보를 아이콘 + 음성 + 색으로 **중복 전달**(색만으로 정보 전달 금지). 큰 터치 타깃으로 작은 손가락도 정확히 누르게.
- **B. "공부"가 아니라 "놀이"로** — 세계관을 스케치북으로 통일하고, 실패를 벌이 아니라 **격려**로 처리(마스코트 응원). 12문제라는 짧은 성취 단위로 도파민 루프를 자주 돌림.
- **C. 부모의 신뢰가 진짜 관문** — 어린이 앱의 채택을 결정하는 건 결국 부모다. **무광고·무결제·무계정·로컬 저장**으로 안심하고 쥐여줄 수 있게 하고, 민감 설정은 부모 게이트 뒤로.
- **D. 접근성·과자극 방지** — 고대비(WCAG AA+), `prefers-reduced-motion` 존중, 소리/음성/음악을 각각 끄고 음량 조절 가능.
- **E. 기술적 실용주의** — 빌드 없는 바닐라 코드로 가볍게, PWA로 앱스토어 심사·수수료 없이 설치형 + 오프라인. 폰트는 로컬 호스팅으로 오프라인에서도 안 깨짐. 컬러 1장 → 자동 흑백/패치로 **적은 에셋으로 콘텐츠 50종 확장**.

## 추가 기능

- 🔥 **데일리 스탬프** — 매일 접속 시 도장, 7일 연속이면 색칠 패치 3개 보너스
- ✨ **콤보 연출** — 3·5·10연속 정답 시 "수리수리 콤보!" 배너 + 사운드
- 🦊 **마스코트 리액션** — 정답/오답에 따라 표정 변화
- 💪 **적응형 약점 집중** — 정답률 낮은 단·연산을 자동 감지해 "약점 연습" 추천
- 🔖 **이어 그리기** — 진행 중인 동물을 홈에서 바로 이어서
- 📚 **그림책 만들기** — 세트 10종 완성 시 Canvas로 합성한 그림책 PNG 저장(공유용)
